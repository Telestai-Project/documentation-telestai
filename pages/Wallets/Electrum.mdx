# Telestai Electrum Wallet Guide

The Telestai Electrum Wallet is a lightweight and fast wallet for Telestai. Unlike the core wallet, it does not require downloading the entire blockchain, making it ideal for everyday use. 

---

## **Key Features**
- **Lightweight and Fast:** Connects to remote nodes, skipping the need to download the blockchain.
- **Secure:** Private keys are stored locally, and the wallet supports advanced security features.
- **Ease of Use:** Intuitive interface for both beginners and experienced users.
- **Cold Storage Support:** Allows for secure offline storage of your TLS funds.

---

## **Download and Installation**

### Step 1: Download the Electrum Wallet
- Access the latest version of the Telestai Electrum Wallet from the [official releases page](https://github.com/Telestai-Project/electrum-telestai/releases).
- Choose the appropriate version for your operating system (Windows, macOS, or Linux).

### Step 2: Install the Wallet
- Follow the installation instructions for your OS:
  - **Windows**: Run the `.exe` installer.
  - **macOS**: Open the `.dmg` file and drag the app to your Applications folder.
  - **Linux**: Extract the tarball and run the executable file.

## **Security and Considerations**
- **Remote Node Dependency:** Relies on external nodes for blockchain access, which may introduce some trust issues compared to the core wallet.
- **Faster Setup:** No need to download the blockchain, making it quick and lightweight.
- **Cold Storage:** Ideal for generating cold storage addresses offline.
